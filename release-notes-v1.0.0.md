# Caffinate v1.0.0

Keep your Mac awake while locked — a small menu bar app that runs the system `caffeinate` command with a clean, modern UI.

## Download

**[Caffinate-macOS.zip](https://github.com/Mbo7682/Caffeinate/releases/download/v1.0.0/Caffinate-macOS.zip)** — unzip, move `Caffinate.app` to Applications (or leave in Downloads), and open it.

If macOS says the app is from an unidentified developer: **right‑click** the app → **Open** → **Open** (once).

## Requirements

- **macOS 14.0 (Sonoma)** or later
- Menu bar only (no dock icon)

## What’s in this release

- **Menu bar caffeinate** — Click the cup icon to start or stop; your Mac stays awake (including when locked).
- **All caffeinate options** — Display, Idle, AC power, User active, Disk, and optional timeout.
- **Notifications** — Alerts when keep-awake starts or stops.
- **Lock screen message** — Optional: show “Caffinate is keeping this Mac awake” on the lock screen (one-time password setup when you turn the option on).
- **Quit** — Quit Caffinate from the popover when you’re done.
